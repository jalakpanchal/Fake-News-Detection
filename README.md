This project detects whether a news article is real or fake using Machine Learning. It also integrates real-time news data from APIs, allowing users to verify live news articles instantly.

**🚀 Features**
✅ Classifies news as Real or Fake
🔄 Fetches real-time news headlines
🧹 Text preprocessing (stopword removal, lemmatization, cleaning)
📊 TF-IDF Vectorization
🤖 ML models (SVM, can use another models too)
🌐 Web interface using Flask
⚡ Fast and scalable pipeline
🌐 Real-Time News Integration

This project uses a news API to fetch latest headlines in real-time and runs them through the fake news detection model.

**🔌 Supported APIs**
_NewsAPI
GNews API_

**💡 How it Works**
Fetch latest news using API
Clean and preprocess text
Convert text using TF-IDF
Predict using trained ML model
Display result (Real / Fake) on UI

**🛠️ Tech Stack**
Python
Pandas, NumPy
Scikit-learn
NLTK / SpaCy
Flask
Requests (for API calls)
